# 🚀 AI Resume Analyzer

An AI-powered web app that analyzes resumes against job descriptions and provides actionable feedback.

## 🔥 Features

* Upload PDF resumes
* Extract text using PDF.js
* Compare resume with job description
* AI-powered analysis (score, strengths, weaknesses)
* Keyword matching for ATS optimization

## 🛠 Tech Stack

* React + TypeScript
* Vite
* OpenRouter (AI API)
* PDF.js

## ⚙️ How it Works

1. Upload resume (PDF)
2. Enter job description
3. AI analyzes and returns structured feedback
